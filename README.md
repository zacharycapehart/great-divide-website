# Great Divide Restaurant Website

A static restaurant website for Great Divide, a farm-to-table restaurant in Denver, CO.

## Project Structure

```
.
├── assets
│   ├── css
│   │   └── styles.css       # All styles, design tokens, responsive layout
│   └── js
│       └── script.js        # Year stamp, smooth scroll, mobile nav, menu tabs, scroll spy
├── docs
│   ├── BRAND_AGENT.md           # Brand standards governance
│   ├── SEO_AGENT.md             # SEO guidelines
│   ├── SECURITY_AGENT.md        # Security checklist
│   ├── CLEAN_CODE_AGENT.md      # Code quality standards
│   ├── UX_AGENT.md              # UX review criteria
│   ├── PERFORMANCE_AGENT.md     # Performance guidelines
│   └── CONTENT_ACCURACY_AGENT.md # Content review standards
├── index.html                   # Single-page restaurant site
└── README.md
```

## Features

- Sticky header with mobile hamburger navigation
- Hero section with call-to-action buttons
- About section with farm partnership stats
- Tabbed menu section (Starters, Mains, Desserts, Drinks)
- Visit section with hours, address, and contact
- Responsive down to small mobile screens
- Accessible: skip link, ARIA roles, keyboard navigation, focus styles
- Smooth scroll, active nav highlighting, and dynamic copyright year

## Run Locally

Open `index.html` in any modern browser — no build step required.

## Deployment

This site is a static HTML/CSS/JS project and can be deployed to any static host (GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.).
