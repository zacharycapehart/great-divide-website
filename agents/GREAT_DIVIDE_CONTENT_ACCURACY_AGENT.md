# Great Divide Content Accuracy Agent

## Purpose
Ensure that all content on the Great Divide Restaurant website is accurate, up-to-date, consistent, and free from errors that could mislead guests or damage the restaurant's reputation.

---

## Content Areas to Audit

### 1. Restaurant Name & Identity
- [ ] Restaurant name is spelled consistently: **Great Divide Restaurant** (not "Great Divide" alone, not "The Great Divide Restaurant").
- [ ] Tagline is consistent across all sections: *Farm to Table*.
- [ ] Brand voice is consistent throughout (see Brand Agent).

### 2. Address
- **Current placeholder:** `123 Main Street, Your City`
- [ ] Real address is filled in before going live.
- [ ] Address format matches the official USPS/postal format.
- [ ] Address is consistent with Google Business Profile, Yelp, and other listings.
- [ ] Google Maps link uses the correct address.

### 3. Phone Number
- **Current placeholder:** `(555) 123-4567`
- [ ] Real phone number is filled in before going live.
- [ ] Phone number is formatted consistently (e.g., `(555) 123-4567`).
- [ ] `tel:` link uses the correct digits without formatting: `href="tel:+15551234567"`.
- [ ] Phone number matches all other online listings.

### 4. Hours of Operation
- **Current placeholder:** `Tue–Sun 11:00 AM – 10:00 PM`
- [ ] Actual operating hours are confirmed with the restaurant owner.
- [ ] Hours reflect current seasonal or temporary changes.
- [ ] Closed days are explicitly stated.
- [ ] Holiday hours policy is mentioned if applicable.
- [ ] Time format is consistent (12-hour with AM/PM preferred for US audiences).

### 5. Menu Content
- [ ] Menu items listed match actual dishes served.
- [ ] Seasonal items are noted as seasonal.
- [ ] Prices (if listed) are accurate and current.
- [ ] Allergen and dietary information is accurate (gluten-free, vegan, etc.).
- [ ] Menu is updated whenever dishes change.

### 6. Reservation & Booking
- [ ] Reservation link/phone/system is working and correct.
- [ ] If using a third-party system (OpenTable, Resy, etc.), the embed or link is current.
- [ ] No references to discontinued booking methods.

### 7. Legal & Copyright
- [ ] Footer copyright year is dynamically updated (via JavaScript).
- [ ] "All rights reserved" statement is accurate.
- [ ] No copyrighted images used without license.
- [ ] Privacy policy linked if any tracking/forms are present.

| Frequency | Review Item |
