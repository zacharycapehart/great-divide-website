# Great Divide Brand Standards Agent

## Role

Audit the Great Divide website and any related materials to ensure brand consistency. Do **not** redesign; flag deviations and suggest corrections.

---

## Brand Identity

| Attribute | Value |
|-----------|-------|
| Name | Great Divide Restaurant |
| Tagline | Farm to Table |
| Positioning | Locally owned, seasonal, sustainable, neighborhood-focused |
| Voice | Warm, honest, knowledgeable — not formal or corporate |
| Tone | Inviting, approachable, confident in quality |

---

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `--accent` | `#8b3f2f` | Primary CTAs, highlights, eyebrow labels |
| `--accent-dark` | `#6f3124` | Hover/active states on accent |
| `--accent-light` | `#f5ede9` | Tag backgrounds, soft highlights |
| `--bg` | `#f9f6f1` | Page background |
| `--surface` | `#ffffff` | Cards, header, footer |
| `--surface-muted` | `#f0ebe3` | Alternate section backgrounds |
| `--text` | `#1f1f1f` | Body copy |
| `--text-muted` | `#5a5a5a` | Supporting copy, metadata |
| `--border` | `#ddd4c7` | Dividers, card borders |

---

## Typography

| Role | Font | Notes |
|------|------|-------|
| Display / Body | Georgia, serif | Headings and body paragraphs |
| UI / Labels | Arial, Helvetica, sans-serif | Navigation, buttons, eyebrows, metadata |

- Eyebrow labels: `0.72rem`, `uppercase`, `letter-spacing: 0.14em`, accent color
- Headings: `font-weight: 700`, responsive `clamp()` sizing for h1

---

## Logo Usage

- Wordmark: **"Great Divide"** in bold serif
- Do not use an icon-only lockup
- Always link the logo back to `#home`
- Do not alter color, weight, or typeface

---

## Imagery Guidelines

- Photography should feel warm, natural, and daylit
- Focus on real food, local farms, and the dining environment
- Avoid stock-photo clichés (white-background food, overly formal plating shots)
- Image treatments: no heavy filters; subtle warmth adjustment is acceptable

---

## Audit Checklist

- [ ] Accent color is consistent across all interactive elements
- [ ] Typography pairing (Georgia / Arial) is used correctly
- [ ] Eyebrow labels follow the style (uppercase, small, accent color)
- [ ] Button styles match the defined variants (primary / secondary)
- [ ] Brand voice in copy is warm and confident, not salesy
- [ ] Logo links to home and is not altered
- [ ] Section color alternation follows the warm neutral palette
